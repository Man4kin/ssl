Назначение групп доступа см. здесь.

Чтобы настроить права доступа нужно выбрать профиль групп доступа.
Профиль определит права (списком ролей) и возможности ограничения прав по разрезам (видам доступа), например, ограничения по видам Организации, Склады, ...
Виды доступа из профиля будут добавлены в группу доступа автоматически.

Исходя из целей использования ограничений, настраиваемых по имеющимся видам доступа, следует сформировать наименование группы доступа (подробнее в назначении групп доступа).

Чтобы настроить ограничения нужно для каждого вида доступа определить список значений.
Для этого по видам доступа нужно установить для всех значений "Запрещены" или "Разрешены",
а затем задать список значений или групп значений, которые будут исключениями.
