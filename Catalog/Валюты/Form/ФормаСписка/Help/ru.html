Команда Подбор из классификатора позволяет добавить новый элемент справочника Валюты, используя ОКВ. После выбора валюты из ОКВ открывается новый элемент справочника Валюты с заполненными реквизитами Наименование валюты, Код символьный, Код числовой, остальнын реквизиты необходимо заполнить самостоятельно. Если курс выбранной валюты можно загружать с сайта РБК, будет установлен флажок Загружается с сайта РБК.

Команда Создать позволяет добавить новый элемент справочника Валюты вручную. В этом случае все реквизиты элемента необходимо заполнить самостоятельно.

Командой Загрузить курсы валют можно открыть форму Загрузка курсов валют с веб-сайта РБК. Командой Заполнить, можно автоматически подобрать все валюты, у которых установлен флаг Загружается с сайта РБК. Командой Подобрать можно выбрать валюту из списка валют, у которых установлен флаг Загружается с сайта РБК. По команде Загрузить, происходит загрузка курсов валют, на каждый день периода, указанного на форме загрузки курсов.

Поиск валюты на сайте РБК, в процессе загрузки курса, осуществляется по реквизиту Код числовой. В случае использования функционала загрузки курсов с сайта РБК, при вводе нового элемента справочника Валюты рекомендуется пользоваться командой Подбор из классификатора, для корректного заполнения указанного реквизита.
