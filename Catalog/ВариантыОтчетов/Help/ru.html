Вариант отчета

Вариант отчета - это настройка отчета, которая содержит описание группировок, отборов, условных оформления и т.п.

Работа с вариантом отчета

В данной справке рассматриваются аспекты работы со списком вариантов отчетов. Вопросы связанные с настройкой и изменением варианта отчета в данной справке не рассматриваются. Для получения наиболее полная информация по настройке вариантов отчета обратитесь к документации по конфигурированию и администрированию 1С:Предприятие версии 8.2.

Создание варианта отчета

Для того, что бы создать новый вариант отчета требуется в соответствующем отчете настроить требуемый вариант отчета и  нажать на кнопку сохранить вариант отчета.

Удаление варианта отчета

Удаление отчет можно производить непосредственно из формы сохранения настроек варианта отчета, а так же пометить вариант отчета на удаление непосредственно из списка вариантов отчета.

Изменение варианта отчета

Изменить вариант можно из формы отчета. Подготовив нужные настройки выбрать пункт меню "Сохранить вариант".

Пользовательские настройки варианта отчета

При работе с отчета требуется изменять настройку варианта отчета. На пример, отключать не нужные группировки, устанавливать отбор, включать или отключать оформление в отчете, говоря другими словами производить настройки, которые не влияют на закономерность расположения данных в отчете. Такие настройки варианта называются пользовательскими и предназначены для того, что бы пользователь не прибегая к настройке варианта отчета смог персонализировать настройку варианта отчета.

Работа с пользовательскими настройками

Вопросы изменения настроек пользователя в данной справке не рассматриваются. Для получения наиболее полной информации по настройке отчета обратитесь к справке отчета.
