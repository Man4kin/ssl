Установить компоненту сканирования - устанавливает компоненту сканирования. Если на Вашем компьютере нет сканера, устанавливать компоненту не нужно. Если компонента уже установлена, то вместо этой кнопки отображается версия установленной компоненты.

Показывать диалог сканера - если Истина, перед сканированием будет показан диалог для расширенной настройки сканирования (например, выбор области сканирования на листе бумаги).

Имя сканера - позволяет выбрать используемый сканер из списка доступных устройств, подключенных к компьютеру.

Формат сканированного изображения - позволяет выбрать один из поддерживаемых форматов (BMP, JPG, GIF, PNG, TIF) для изображений, которые будут получены со сканера.

Разрешение - позволяет выбрать разрешение сканирования (DPI). Список доступных разрешений зависит от выбранного сканера.

Цветность - выбор цветности изображений, которые будут получены со сканера (Монохромное, Градации Серого, Цветное).

Поворот изображения - позволяет выбрать один из вариантов поворота изображений после сканирования. Для некоторых моделей сканеров это свойство недоступно.

Размер бумаги - позволяет выбрать размер бумаги, которая помещена в сканер. Для некоторых моделей сканеров это свойство недоступно.

Открыть номера отсканированных файлов  - открывает список последних использованных номеров, из которых автоматически создаются имена отсканированных файлов для каждого владельца (папка файла или Входящий, Исходящий, Внутренний документ)
