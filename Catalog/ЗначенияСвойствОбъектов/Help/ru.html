Значения дополнительных реквизитов и сведений могут выбираться из специального справочника - "Значения свойств объектов".

При этом для каждого дополнительного реквизита или сведения список значений будет свой. Если список значений большой, то может быть удобно создать группы и разнести значения по ним.
