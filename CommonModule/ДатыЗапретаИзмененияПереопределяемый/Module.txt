// Процедура ПередПроверкойЗапретаИзменения позволяет по некоторому
// условию, например, для отказа проверки в некоторых режимах работы
// подсистемы обмена данными (если она используется) или
// для отказа от проверки по статусу документа (открыт/закрыт).
//
// Параметры:
//  Объект       - объект или набор записей (ПередЗаписью или ПриЧтенииНаСервере):
//                 СправочникОбъект
//                 ДокументОбъект
//                 ПланВидовХарактеристикОбъект
//                 ПланСчетовОбъект
//                 ПланВидовРасчетаОбъект
//                 БизнесПроцессОбъект
//                 ЗадачаОбъект
//                 ПланОбменаОбъект
//                 РегистрСведенийНаборЗаписей
//                 РегистрНакопленияНаборЗаписей
//                 РегистрБухгалтерииНаборЗаписей
//                 РегистрРасчетаНаборЗаписей
//
//  СтандартнаяОбработка - Булево, начальное значение Истина,
//                 если присвоить значение Ложь,
//                 тогда проверка выполняться не будет
//
//  ПроверятьПриОбменеДанными - Булево, начальное значение Ложь,
//                 если присвоить значение Истина,
//                 тогда проверка выполняться не будет,
//                 когда Объект.ОбменДанными.Загрузка = Истина
//
Процедура ПередПроверкойЗапретаИзменения(Объект, СтандартнаяОбработка, ПроверятьПриОбменеДанными) Экспорт
	
КонецПроцедуры

// Процедура УстановитьФормуНастройкиЗапретаИзмененияПриОбменеДанными используется

// при выполнении обмена данными

//  ПолноеИмяФормы - Строка - передается пустая строка, которую можно заполнить,
//                   если не заполнена, значит команда не используется
//  ЗаголовокКомандыОткрытияФормы - Строка - передается строка
//                   НСтр("ru = 'Использование при обмене данными'"), которую можно изменить
//
Процедура УстановитьФормуНастройкиЗапретаИзмененияПриОбменеДанными(ПолноеИмяФормы, ЗаголовокКомандыОткрытияФормы) Экспорт
