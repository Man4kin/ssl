Настройка версионирования документов и справочников. Влияет на сохранение версий объекта при изменении (создании).

  • Не версионировать - версии объекта не сохраняются
  • Версионировать всегда - версии объекта сохраняются при любом изменении (создании)
  • Версионировать при проведении - версии сохраняются только при проведении; настройка применяется только для документов, для которых разрешено проведение

Допускается установка настроек версионирования для группы объектов.
