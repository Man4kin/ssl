Форма предназначена для монитора и контроля выполнения обмена данными. В форме отображаются все обмены данными, созданные в информационной базе и статусы их выполнения.

Если пользователю назначена роль "Добавление изменение обменов данными", то в контекстном меню формы будет доступна кнопка "Создать".
По кнопке "Создать" открывается меню возможных настроек обмена данными с различными конфигурациями, определенных в системе. При выборе пункта меню будет открыт помощник создания обмена данными.

По кнопке "Выполнение обмена данными" открывается меню с командами выполнения обмена в автоматическом или интерактивном режиме для текущей информационной базы, выделенной в списке. Можно быстро перейти к журналу регистрации для просмотра событий обмена данными. Все эти команды также доступны из контекстного меню списка.
